# Правила контента и источников

- Для каждого события обязателен заголовок, тип, описание и экспедиция.
- Для каждой геоточки обязательны координаты и пометка точности при необходимости.
- Для каждого источника нужно хранить название, тип, происхождение, дату или период и ссылку на архив/издание, если она известна.
- Цитаты должны быть привязаны к конкретному событию.
