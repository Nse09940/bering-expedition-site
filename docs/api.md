# API

MVP API закрывает публичный каталог, карту и будущую CMS.

## Публичные endpoints

- `GET /api/health/` - проверка доступности сервиса.
- `GET /api/expeditions/` - опубликованные экспедиции.
- `GET /api/expeditions/{slug}/` - экспедиция, этапы, маршруты, события, цитаты и медиа.
- `GET /api/events/` - события с фильтрами `expedition_slug`, `event_type`, `q`.
- `GET /api/sources/` - источники с фильтрами `source_type`, `q`.
- `GET /api/map-data/` - агрегированные данные для карты.
- `GET /docs` - OpenAPI-документация FastAPI.

## CMS-контракт

Админка должна выполнять CRUD для экспедиций, этапов, событий, геоточек,
маршрутов, источников и медиа. Для публикации обязательны:

- статус `published`;
- источник для каждой цитаты, карты и изображения;
- alt-текст и credit для медиа;
- примечание о точности координат;
- ссылка на архив, каталог или издание источника.
